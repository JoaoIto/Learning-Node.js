# Desafio 1️⃣

O desafio 1, (referente sempre ao módulo correspondente, no caso "module1".), consistia em utilizar os conhecimentos relativos a módulo e process.argv!

## Sobre o desafio:

- **1°: Crie um arquivo de módulo de uma função que construção um `process.argv` de flags chamadas no node.**

- **2°: Exporte a função para um arquivo que execute a função em node, e consiga entender cada argumento.**

- **3°: Faça o `console.log("")`, para cada index do argv e assim as definidas flags desses argv.**
